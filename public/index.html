A nicer way to share Blueprints

Klee enables simple sharing of Blueprints created in Unreal Engine 4 or Unreal Engine 5.

No Screenshots needed!

Copy and paste your Blueprints directly from Unreal Engine into any modern webbrowser without the need of taking and uploading screenshots.

No more recreating nodes you found on a screenshot. Select it and copy & paste it like it was text-based code.

Simple integration

Klee is written in TypeScript and compiled to JavaScript. All the magic without any additional dependencies. 

                                
<canvas class="klee"> YOUR COPIED BP CODE HERE </canvas>
...
<script src="klee.min.js"></script>

We already support

  • Custom events nodes
  • Custom function nodes
  • Function nodes (construction, return node, ...)
  • Call function nodes (math-, string-, array nodes, ...)
  • Input event nodes (keyboard, touch, ...)
  • Variable nodes (getter, setter)
  • Reroute nodes (knots)
  • Dynamic cast nodes
  • Flow control nodes (loops, sequences, branches, ...)
  • Nodes for structs (make, break, ...)
  • Nodes to manage timers
  • Macro nodes
  • Switch nodes
  • Timeline nodes
  • Comment blocks

Share everywhere

Klee lets you share Blueprints on your own Website like you would share text-based code. Share Blueprints in your forum, your tutoring site or your personal Blog.
